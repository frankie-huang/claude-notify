# Change: 为 /new 指令添加工作目录选择卡片

## Why

当用户执行 `/new` 指令时，如果没有指定目录参数（`--dir=`）或者无法从回复消息获取工作目录时，当前系统只能返回文本错误提示，要求用户手动输入完整命令。这种方式用户体验不佳，需要用户重新记忆并输入完整的命令格式。

通过提供一个交互式飞书卡片，让用户可以从常用工作目录列表中选择，可以显著提升用户体验，降低使用门槛。

## What Changes

- **新增工作目录选择卡片**：当 `/new` 指令缺少工作目录时，发送飞书卡片而非纯文本错误
- **常用目录动态收集**：系统自动从历史会话记录中收集高频使用的工作目录
- **交互式下拉菜单**：卡片包含可输入的下拉菜单组件，支持选择或手动输入目录
- **一键提交按钮**：用户选择目录后点击按钮即可提交会话创建请求
- **卡片动态更新**：提交成功后自动更新卡片状态，显示会话创建成功信息并移除按钮

## Impact

- Affected specs: `feishu-command` (扩展能力), `card-callback` (新增目录选择卡片回调处理)
- Affected code:
  - `src/server/handlers/feishu.py` - 修改 `/new` 指令处理逻辑
  - `src/server/handlers/callback.py` - 新增卡片选择回调处理
  - `src/server/services/session_store.py` - 新增目录使用历史记录功能
